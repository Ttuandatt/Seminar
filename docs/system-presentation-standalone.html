02 — Kiến trúc hệ thống

Modular Monolith
+ Layered Architecture (N-tier)

Toàn bộ backend chạy trên 1 NestJS process duy nhất, bên trong chia thành các module độc lập với ranh giới rõ ràng. Các module dùng chung 1 database và giao tiếp qua dependency injection trong cùng process.

Tại sao chọn Modular Monolith?

🎯

Phù hợp quy mô

Seminar/startup, team 2-5 người. Microservice chỉ cần khi team 10+ và > 10.000 CCU

⚡

Hiệu suất cao

Giao tiếp giữa modules = gọi hàm trong process. Không qua HTTP/gRPC/message queue

💰

Chi phí thấp

1 server, 1 DB, 1 deploy. Không cần service discovery hay container orchestration

07 — Dịch thuật thông minh

NER + Translation Memory

"Hai Bà Trưng" KHÔNG được dịch thành "Two Lady Trung". Hệ thống sử dụng Named Entity Recognition để nhận diện địa danh trước khi dịch.

Input (Tiếng Việt)
Quán nằm trên đường Hai Bà Trưng, gần chợ Bến Thành
Output (English)
The restaurant is on Hai Bà Trưng Street, near Bến Thành Market
NER Tags
PROPER_NOUN → Giữ nguyên phiên âm
LANDMARK → Dịch từ generic ("chợ" → "Market")
So sánh
Two Lady Trung Street ✗ Sai
Hai Bà Trưng Street ✓ Đúng
TTS Pronunciation
"Hai Ba Trung" → /haːj ɓaː ʈɨŋ/
08 — Offline & Fallback

Cơ chế phòng ngự
4 lớp

Khi user đang dùng mà bị mất kết nối internet, hệ thống tự động chuyển đổi qua các lớp fallback.

L1
Cache Hit
Audio/data đã cache → phát bình thường. User không biết bị mất mạng.
✅ Transparent
L2
Stale Data
Cache hết hạn nhưng vẫn còn data cũ → dùng data cũ + tag "Offline".
⚠ Stale
L3
Offline Database (SQLite)
SQLite local đã sync → đọc POI text. Map chuyển sang PMTiles offline.
⚠ Text only
L4
Graceful Degradation
Không có data → bản đồ trống + "Không có kết nối" + [Retry].
❌ Limited
09 — Chế độ bản đồ

Cloud / Offline / Hybrid

Tự động chuyển đổi giữa 3 chế độ dựa trên trạng thái kết nối và cấu hình device.

Default

☁ Cloud

Google Maps API — chất lượng cao nhất, real-time traffic

+ Chất lượng cao nhất
- Cần internet ổn định
- Tốn Google Maps API fee
Pre-download

💾 Offline

PMTiles local — download 50-100MB cho khu vực Q4

+ Không cần internet
- Tốn 50-100MB storage
- Không real-time
Khuyến nghị

🔄 Hybrid

Cache tiles đã xem + PMTiles fallback. Tự chuyển khi mất mạng.

+ Cân bằng tốt nhất
- Phức tạp implementation
