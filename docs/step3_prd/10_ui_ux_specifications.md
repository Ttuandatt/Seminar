## 5. UI States

### 5.1 Loading States

| Component | State |
|-----------|-------|
| Page load | Skeleton screens |
| Data fetch | Spinner + "Loading..." |
| Button action | Spinner inside button, disabled |
| Image load | Placeholder â†’ fade in |
| Audio load | "Loading audio..." |

### 5.2 Empty States

| Screen | Message | CTA |
|--------|---------|-----|
| POI List (Admin) | "No POIs yet" | "Create your first POI" |
| POI List (Shop Owner) | "No POIs yet" | "Create your first POI" |
| Tour List | "No Tours" | "Create a Tour" |
| Search results | "No results for '...'" | "Clear filters" |
| Offline | "You're offline" | "Retry" |

### 5.3 Error States

| Error | Display | Action |
|-------|---------|--------|
| Form validation | Inline red text below field | Fix and resubmit |
| API error | Toast notification | Retry option |
| 404 | Full page with illustration | Go back / Home |
| Network error | Banner "No connection" | Auto-retry |
