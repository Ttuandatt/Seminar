## 8. Interaction Patterns

### 8.1 Pull to Refresh
- Tourist app: POI list, Tour list, Favorites
- Pull down â†’ spinner â†’ reload data

### 8.2 Infinite Scroll vs Pagination
- **Admin/SO tables:** Pagination (10/25/50 per page)
- **Tourist lists:** Infinite scroll

### 8.3 Swipe Actions (Mobile)
- Favorites list: Swipe left â†’ Remove from favorites
- History list: View only, no swipe actions

### 8.4 Confirmation Patterns
- **Single confirm:** Delete Tour, Unpublish POI
- **Double confirm:** Delete POI (warning cascade â†’ confirm dialog)
- **No confirm:** Save draft, Toggle favorite, Language switch

### 8.5 Auto-save
- POI form: Auto-save draft every 30 seconds (Admin)
- No auto-save for Shop Owner (explicit Save only)
